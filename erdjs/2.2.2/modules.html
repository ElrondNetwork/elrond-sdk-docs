appendArguments

  • appendArguments(to: string, args: Argument[]): string
    • Defined in src/smartcontracts/argument.ts:118

    Appends Argument objects to a given string. The resulted string is to be used for preparing Smart Contract calls (or deployments). In general, this function should not be used directly. It is used by TransactionPayload builders (such as {@link ContractCallPayloadBuilder}), under the hood.

    let data = appendArguments("transferToken", [Argument.pubkey(alice), Argument.number(42)]);
let payload = new TransactionPayload(data);
